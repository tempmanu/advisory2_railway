Smart Crop Advisory

Empowering farmers with AI-based soil analysis for better yields.

📋 About Project

This system uses soil data (type, pH, water availability) through a rule-based AI engine to generate personalized fertilizer recommendations. Built as part of the SOP project to support farmers in Telangana.
